# Program1

## Описание
Этот проект представляет собой многопоточное приложение на C++, которое обрабатывает пользовательский ввод и отправляет результаты на сервер. Пользователь вводит строку, содержащую только цифры, программа сортирует ее в обратном порядке и модифицирует цифры, а затем отправляет сумму цифр на сервер.

### Установка

1. В директории проекта создайте директорию build и перейдите в неё:
    ```bash
    mkdir build
    cd build
    ```

2. Соберите проект:
    ```bash
    cmake ..
    make
    ```

### Запуск проекта

1. Запустите приложение:
    ```bash
    ./Program1
    ```

2. Введите строку, содержащую только цифры и не превышающую 64 символа. Строка будет обработана и результат будет отправлен на сервер, работающий на порту 12345.

## Использование
После запуска приложения:
1. Пользователь вводит строку.
2. Строка сортируется в обратном порядке.
3. Каждая четная цифра заменяется на 'KB'.
4. Вычисляется сумма всех цифр.
5. Сумма отправляется на сервер.
