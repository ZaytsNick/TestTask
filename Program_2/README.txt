# Program2

## Описание
Этот проект представляет собой простой TCP-сервер на C++, который принимает данные от клиента, проверяет, делится ли полученное число на 32, и выводит результаты проверки в консоль.

### Установка

1. В директории проекта создайте директорию build и перейдите в неё:
    ```bash
    mkdir build
    cd build
    ```

2. Соберите проект:
    ```bash
    cmake ..
    make
    ```

### Запуск проекта

1. Запустите приложение:
    ```bash
    ./Program2
    ```

Сервер будет слушать входящие соединения на порту 12345.

## Использование

1. Клиент должен подключиться к серверу на порту 12345.
2. Клиент отправляет строку, содержащую число.
3. Сервер принимает строку, выводит ее в консоль и проверяет, строка больше 2 символов и делится ли число на 32.
4. Результат проверки выводится в консоль.
